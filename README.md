## *Objective:*
The aim of the project is a temporary email server with the same layout as gmail

## *See live:*
https://danilosoftwares.github.io/gmail-fake/

## *Installing*

```sh
git clone https://github.com/danilosoftwares/gmail-fake
cd gmail-fake
npm install
npm start
```
